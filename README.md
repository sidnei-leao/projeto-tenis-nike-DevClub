# Projeto Tênis Nike

Projeto de uma página interativa de tênis Nike desenvolvido com HTML, CSS e JavaScript.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎨 Funcionalidades

- Troca de cor do fundo
- Alteração da imagem do tênis
- Layout responsivo
- Animação flutuante do produto
